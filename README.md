# Met Photo Library

Met photo library helps the Met police to store photos of suspects involved in a criminal case, and photos of asylum seekers.

## Suspect Photos

- Front View
- Left Profile
- Right Profile
- Name
- CPR number
- Current Addres
- Physical Description
  1. race
  2. age
  3. height
  4. weight
  5. eye colour
- Crime Type
- Case Number
